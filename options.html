Comment obtenir votre clé API ?

  1. Rendez-vous sur dashboard.audd.io
  2. Créez un compte gratuit ou connectez-vous
  3. Dans le tableau de bord, vous trouverez votre clé API (Token)
  4. Copiez la clé et collez-la ci-dessous
La clé API est nécessaire pour identifier les musiques. Elle reste stockée localement dans votre navigateur.
Note : Le service gratuit d'AudD permet d'identifier jusqu'à 10 chansons par jour. Pour plus d'identifications, vous pouvez passer à un plan payant sur leur site.
